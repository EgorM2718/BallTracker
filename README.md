# BallTracker
Отслеживание теннисного мяча в реальном времени с помощью классических алгоритмов компьютерного зрения

Целью данного проекта было разобраться с базовыми способами обработки изображений и основными алгоритмами компьютерного зрения.

### Как эта программа работает?
1. Создается цветовой диапазон, по которому для каждого кадра будет создаваться маска;
2. Полученная маска обрабатывается с помощью морфологических операций;
3. В обработанной маске ищутся контуры, вокруг каждого из которых описывается окружность;
4. Радиус этой окружности сравнивается с пороговым значением, и в зависимости от этого принимается решение, является ли данный объект теннисным мячом или нет.
